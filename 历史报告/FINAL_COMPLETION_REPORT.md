## 🚀 立即开始使用

### 5分钟快速启动

```bash
# 1. 克隆项目
git clone <repository>
cd agentbus

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置本地模型（Ollama）
curl -fsSL https://ollama.ai/install.sh | sh
ollama pull codellama:7b
ollama serve &

# 4. 启动AgentBus
python -m agentbus.core.app --mode web --host 0.0.0.0 --port 8000

# 5. 访问Web界面
# 浏览器打开: http://localhost:8000
```
